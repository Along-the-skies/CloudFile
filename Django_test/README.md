# 🌩️ CloudFile

CloudFile is a Django-powered web application for **cloud-based file storage and sharing**.  
👉 Try it live: [CloudFile by VASU](https://bit.ly/CloudFile-byVASU)

---

## 🚀 What is CloudFile?
CloudFile lets you:
- 📂 Upload files securely
- 🔗 Generate shareable links
- 👤 Manage files with authentication
- 🌐 Access your files from anywhere

---

## 🖥️ Live Demo
Visit the hosted version here:  
🔗 [bit.ly/CloudFile-byVASU](https://bit.ly/CloudFile-byVASU)

---

## 🛡️ Security
- Files are protected with Django’s authentication system.
- CSRF and host settings are properly configured.
- HTTPS recommended for production deployments.
